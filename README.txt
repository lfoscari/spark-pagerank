Working dataset: https://www.kaggle.com/datasets/bwandowando/ukraine-russian-crisis-twitter-dataset-1-2-m-rows.

The task is to implement from scratch a system ranking tweet users, considering
the relation R = {(u, v) | user u has mentioned user v in a tweet} and
computing, e.g.,  the PageRank index.
